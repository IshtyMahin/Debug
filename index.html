How does JS work?

IDE এ যে সকল code লিখা সেগুলাকে source code বলা হয়। source code গুলাকে একটি compiler নামক একটি প্রোগ্রামের মাধ্যমে পাস করা হয়, যা এটিকে machine code এ অনুবাদ করে । এবং সেগুলার মাধ্যমে কোড গুলো মেশিনটি বোঝে এবং সম্পাদন করতে পারে। কিন্তু জাভাস্ক্রিপ্টের compiler নেই। ফলে, ব্রাউজারের interpreter জাভাস্ক্রিপ্ট কোড গুলো্কে প্রতিটি লাইন ব্যাখ্যা করে এবং এটি চালায়।

Local storage vs Session storage

ওয়েব স্টোরেজ ব্যবহারকারীর ব্রাউজারের মধ্যে স্থানীয়ভাবে ডেটা সংরক্ষণ করতে পারে।ওয়েব স্টোরেজ ২ ধরনের।যথাঃ ১।Local strorage ২।Session storage Local strorage ও Session storage প্রায় অনুরূপ। একটি Session storage যতক্ষণ ট্যাব বা ব্রাউজার খোলা থাকে ততক্ষণ স্থায়ী হয়। যদি page reload বা ব্রাউজার বন্ধ করলে Session storage সাথে সাথে চলে যাই। কিন্তু Local strorage browser এ স্থায়ীভাবে থেকে যাই। যতক্ষণ পর্যন্ত আমরা ব্রাউজার থেকে নিজেরা না কাটি। Local strorage এর কোনো expiration date নাই।ব্রাউজার বন্ধ করে দিলে ও Local strorage থেকে যাই।
